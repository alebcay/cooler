Cooler

Take your Homebrew, Cask, and MacPorts packages with you wherever you go

    Get a list of your configured taps and Homebrew, Cask, and MacPorts packages with one command on your home machine.

    $ cooler export myExport

    Copy the resulting Cooler bundle file: ./myExport.cb to the target machine.

    Install the packages on your target system.
    $ cooler import myExport

    Uninstallation is easy: there's only one file to delete.

    $ rm /usr/local/bin/cooler

    Cooler bundles are just tarballs in disguise, so edits are easy to make if needed.

    $ tar xzvf myExport.cb
    x ./myExport/
    x ./myExport/brew.cr
    x ./myExport/cask.cr
    x ./myExport/port.cr
    x ./myExport/tap.cr

    Ship just parts of your environment if you'd like. Likewise, install just parts of a bundle if you'd like.

    $ cooler export myExport --no-brew
    $ cooler import myExport --no-brew
    $ cooler export myExport --no-cask
    $ cooler import myExport --no-cask
    $ cooler export myExport --no-port
    $ cooler import myExport --no-port

    Install Cooler

    curl -Ls https://raw.github.com/alebcay/cooler/master/installcooler | bash

    Paste that at a Terminal prompt.

    Cooler is written completely in shell script for accessibility and simplicity.
